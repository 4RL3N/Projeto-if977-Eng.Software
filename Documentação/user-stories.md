#  User Stories - PART (Plataforma de Acomodação e Residência para Todos)


### Universitários:
```
Como universitário, eu quero procurar alguém para dividir o quarto por meio de uma publicação,
para encontrar pessoas que busquem a mesma coisa e economizar nos custos de moradia.
```

```
Como universitário, eu quero achar moradias com filtros avançados, como preço, localização
e tipo de imóvel, para ter várias opções que atendam às minhas necessidades.
```

```
Como universitário, eu quero poder ter o contato do proprietário ou da pessoa que está oferecendo o
quarto para dividir, para poder negociar diretamente.
```

```
Como universitário, eu quero receber notificações de novas publicações de quartos ou imóveis para dividir,
para não perder a oportunidade de encontrar um lugar.
```

### Proprietários
```
Como proprietário, eu quero publicar minha casa para o público-alvo, para que possa alugar
ou dividir o imóvel com pessoas interessadas.
```

```
Como proprietário, eu quero fornecer meu contato para os interessados no imóvel,
para facilitar a negociação e o fechamento do contrato.
```

### Usuários Gerais
```
Como usuário geral, eu quero poder apagar minha publicação,para remover informações que não são mais relevantes
ou se eu já não estou mais oferecendo o quarto.
```

```
Como usuário geral, eu quero poder visualizar minha publicação, para verificar se todas as informações estão
corretas e atualizadas.
```

### Administrador
```
Como administrador, eu quero aceitar ou recusar postagens publicadas, para garantir que apenas as postagens
que cumprem os critérios e normas do sistema sejam exibidas para os usuários.
```
